# Rsoft.Helpers
Helpers to assist in repetitive day-to-day operations for developers

## Features

* Get properties by reflection
* MD5 converters
* Generic TryCast
* Date and Time methods
    * LastDayInMonth
    * Age
* Email
    * Get Login
    * Get Domain
* Files and Paths
    * RemovePath
    * NetworkPathBase
    * SaveFileAsync
* Numbers and Strings
    * RemoveText
    * RemoveNumbers
* Validation
    * Brasilian Documents
        * Cpf
        * Cnpj
    * CreditCard
		* Check
		* IsBrand
